MiniJava Lexer
======

Mini Java Lexical Analysis

[BNF for MiniJava](https://www.cambridge.org/us/features/052182060X/grammar.html)

# Requirements

``` shell
apt-get install build-essential graphviz flex bison
```

# Usage

``` shell
$ make
$ ./compiler <file>
$ make tree
$ make clean
```
