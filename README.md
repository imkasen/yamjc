MiniJava Compiler
======

Mini Java Compiler Practice.

[BNF for MiniJava](https://www.cambridge.org/us/features/052182060X/grammar.html)

# Requirements

``` shell
apt install build-essential graphviz flex bison
```

# Usage

``` shell
$ make
$ ./compiler <file>
$ make tree
$ make clean
```
